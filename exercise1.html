Exercise 1

We're now going to use your newly developed skills to make a Force Calculator!.
Let's just start with one of the famous Newton's Laws of Motion: F=ma!

1. Create an action to calculate F, given m and a. (remember, use input and output parameters!)
2. Repeat for m using F,a
3. Repeat for a using F,m
4a. In the main action, make it interactive
4b. Ask the user if they want to calculate Force, Mass, or Acceleration
4c. Then ask for each input, and show them the answer!
