About

Title

Touch Develop programming course for secondary school students.

Theme

Coding exercises in conjunction with physics related topics.
The aim of these learning exercises is to introduce students to the world of programming. They will work with a language called touch develop whereby they will learn and eventually write basic code. The programs they work with will implement some of the core principles associated with physics.

Scope

The course will contain 12 exercises ranging from beginner exercises to intermediate level. The purpose of this course is intended for students aged 11+.

Learning Outcomes

1. Understand the importance of programming and its uses
2. Confidently use the Touch Develop Programming language
3. Understand the key fundamental concepts of both programming and key stage 2/3 physics
4. Hopefully want to engage with programming further into their academics
Content

Tutorials 1-11: The initial set of exercises and tutorials will cover the main programming concepts such as variables, loops and if statements.
The next set of tutorials will integrate physics problems and take what students have learnt from the previous lessons.
The final tutorials will elaborate on what students already know and use these skills for the more complex coding scenarios.
Alongside the tutorials will be exercise questions for students to complete on their own or within a group for later assessment.

Resources Required

Students will require a computing device (i.e. computer, phone, tablet etc.) with online connectivity. They will need to access the Touch Develop Website and access the course accordingly.
